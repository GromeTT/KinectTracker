bool SkeletonAnalyzer::update ( const SkeletonDataPtr  skeleton,
const unsigned int  timestamp 
)

This function updates the tracking results.

For this purpose two bounding boxes will be constructed from skeleton.
The second BoundingBox surrounds the whole body, so that one can determine the basic pose of the tracked person. Furthermore the BoundingBox can be used to find out in which area of the image data the person is standing.
