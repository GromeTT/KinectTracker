TreeModel::createItem Creates an QStandardItem displaying the data from value.

The flags Qt::ItemIsSelectable and Qt::ItemIsEnabled are always set. If metaProperty is writable and editable is set to true the flag Qt::ItemIsEditable will be set and the data is stored in Qt::EditRole. Otherwise the data will be put in the Qt::DisplayRole slot.
