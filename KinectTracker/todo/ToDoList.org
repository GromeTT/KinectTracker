     + Farbräume
       + HSV ( Hue Saturation Value):
       	 + Hue: Farbwert
       	 + Saturation: Farbsättigung
	   - 0% Neutralgrau
	   - 50% Wenig gesättigte Farbe
	   - 100% Gesättigte Farbe / reine Farbe
       	 + Value: Hellwert bzw. Dunkelstue
	   - 0% keine Helligkeit
	   - 100% volle Helligkeit
